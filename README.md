# Nabha Care Telemedicine Platform

A multilingual telemedicine platform designed to improve healthcare accessibility in rural areas.

## Features

- Patient Registration & Login
- Doctor Dashboard
- Hospital Management
- Appointment Booking
- Video Consultation
- Medical Records Management
- Pharmacy Module
- Multilingual Support
- Real-time Communication

## Technologies Used

- Next.js
- JavaScript
- HTML
- CSS
- Node.js
- MongoDB
- WebRTC
- Git & GitHub

## Project Objective

To provide accessible healthcare services for rural communities through a digital telemedicine platform supporting consultation, appointments, and healthcare management.
